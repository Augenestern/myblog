Day.js 是一个轻量的处理时间和日期的 JavaScript 库
<!-- more -->

### 安装Day.js
``` bash
npm install dayjs --save
```
### 使用
``` bash
dayjs('2018-08-08') // 解析
dayjs().format('{YYYY} MM-DDTHH:mm:ss SSS [Z] A') // 展示
dayjs()
  .set('month', 3)
  .month() // 获取
dayjs().add(1, 'year') // 处理
dayjs().isBefore(dayjs()) // 查询
```
